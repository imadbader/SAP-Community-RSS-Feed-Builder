URL Pattern Reference
⚠️ Known limitation: SAP native tag RSS feeds are imprecise
The SAP Community platform's /rss/search?q=tags:[name] endpoint performs a loose keyword search — not a strict managed tag filter. Posts may appear in the feed without having the tag, and posts with the tag may be missing. For exact tag filtering, the open-source rss-scn.marianzeis.de API can be used manually — but requires the tag title to exactly match the SAP Community managed tag registry. See the URL Reference tab for the format.

⚠️ Tag RSS — SAP Native (imprecise)

Keyword search, not strict tag match:
/khhcw49343/rss/search?q=tags:[Tag%20Name]

Example:
…?q=tags:SAP%20AI%20Core

✅ Tag RSS — Accurate (third-party, manual)

For exact managed tag filtering, use rss-scn.marianzeis.de — but only with tag titles verified against the SAP Community managed tag registry, as the API does a strict exact-match lookup.

rss-scn.marianzeis.de/api/messages?conversation.style=blog&managedTag.title=[Exact Tag Title]&feeds.replies=false

conversation.style is required (blog or qanda).
